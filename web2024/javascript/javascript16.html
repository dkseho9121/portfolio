01. slider Effect

スライド効果の基本的な効果：トランジション効果

  • データを保存する: 変数
  • 要素オブジェクト: メソッド: querySelector( ), querySelectorAll( )
  • 関数: メソッド: setInterval( ): 各呼び出しの間に固定された時間が遅れて関数を繰り返し呼び出すか、コードスニペットを実行します。
  • CSS: opacity: 要素の不透明度を設定します(0〜1の間の数字のみ入力可能)。
  • ライブラリ: GSAP: gsap.to( )
  • ライブラリ: jQuery: $( )
